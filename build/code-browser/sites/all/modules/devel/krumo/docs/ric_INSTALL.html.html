  14. 14                      SETUP: How to install Krumo ?
  15. 15 
  16. 16 ------------------------------------------------------------------------------
  17. 17 
  18. 18 In order to use Krumo you have to put it on your (development) server, and 
  19. 19 include it in your script. You can put it somewhere in the INCLUDE_PATH, or 
  20. 20 specify the full path to the &quot;class.krumo.php&quot; file.
  21. 21 
  22. 22 You have to modify the &quot;krumo.ini&quot; file too. It is the configuration file for 
  23. 23 Krumo. The first option is choosing a skin:
  24. 24 
  25. 25  [skin]
  26. 26  selected = &quot;orange&quot;
  27. 27 
  28. 28 The value for this setting has to be the name of one of the sub-folders from the 
  29. 29 &quot;Krumo/skins/&quot; folder. If the value provided for the skin results in not finding 
  30. 30 the skin, the `default` skin will be used instead.
  31. 31 
  32. 32 The second option is used to set the correct web path to the folder where Krumo 
  33. 33 is installed. This is used in order to make the images from Krumo's CSS skins 
  34. 34 web-accessible.
  35. 35 
  36. 36  [css]
  37. 37  url = &quot;http://www.example.com/Krumo/&quot;
  38. 38 
  39. 39 So far those two are the only configuration options.
  40. 40 
  41. 41 All the CSS files (&quot;skin.css&quot;) from the &quot;Krumo/skins/&quot; sub-folders must have the 
  42. 42 proper permissions in order to be readable from Krumo. Same applies for 
  43. 43 &quot;krumo.ini&quot; and &quot;krumo.js&quot; files.
