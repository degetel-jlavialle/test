  13.  13 /**
  14.  14  * Allow modules to define RDF mappings for field bundles.
  15.  15  *
  16.  16  * Modules defining their own field bundles can specify which RDF semantics
  17.  17  * should be used to annotate these bundles. These mappings are then used for
  18.  18  * automatic RDFa output in the HTML code.
  19.  19  *
  20.  20  * @return
  21.  21  *   A list of mapping structures, where each mapping is an associative array:
  22.  22  *   - type: The name of an entity type (e.g., 'node', 'comment', and so on.)
  23.  23  *   - bundle: The name of the bundle (e.g., 'page', 'blog', or
  24.  24  *     RDF_DEFAULT_BUNDLE for default mappings.)
  25.  25  *   - mapping: The mapping structure which applies to the entity type and
  26.  26  *     bundle. A mapping structure is an array with keys corresponding to
  27.  27  *     existing field instances in the bundle. Each field is then described in
  28.  28  *     terms of the RDF mapping:
  29.  29  *     - predicates: An array of RDF predicates which describe the relation
  30.  30  *       between the bundle (RDF subject) and the value of the field (RDF
  31.  31  *       object). This value is either some text, another bundle, or a URI in
  32.  32  *       general.
  33.  33  *     - datatype: Is used along with 'callback' to format data so that it is
  34.  34  *       readable by machines. A typical example is a date which can be written
  35.  35  *       in many different formats but should be translated into a uniform
  36.  36  *       format for machine consumption.
  37.  37  *     - callback: A function name to invoke for 'datatype'.
  38.  38  *     - type: A string used to determine the type of RDFa markup which will be
  39.  39  *       used in the final HTML output, depending on whether the RDF object is a
  40.  40  *       literal text or another RDF resource.
  41.  41  *     - rdftype: A special property used to define the type of the instance.
  42.  42  *       Its value should be an array of RDF classes.
  43.  43  *
  44.  44  *       @ingroup rdf
  45.  45  */
